# Budget Rules

작성일: 2026-07-05

이 문서는 예산 관리 1차 기능화 기준이다.

## 계산 기준

- 배정 예산: 부서별 회계 기간에 승인된 총 예산이다.
- 사용 금액: 승인 완료 또는 지급 예정 이상으로 확정된 결제 요청 금액의 합계다.
- 잔액: `배정 예산 - 사용 금액`으로 계산한다.
- 사용률: `사용 금액 / 배정 예산 * 100`을 반올림한 정수 퍼센트로 표시한다.

## 상태 판정

- 정상: 사용률 80% 미만
- 주의: 사용률 80% 이상 100% 미만
- 초과: 사용률 100% 이상

## 예산 조정

- 조정은 `BudgetAdjustment` 이력으로 부서, 금액, 사유, 요청자, 승인 필요 여부, 상태, 요청일시, 적용일시를 남긴다.
- 10,000,000원 이상 조정은 승인 대기 상태로 저장하고, 예산 배정액은 승인 전까지 변경하지 않는다.
- 10,000,000원 미만 증액 조정은 같은 DB 트랜잭션에서 배정 예산과 잔액을 같은 금액만큼 증가시키고 감사 로그를 남긴다.
- 감액 조정은 배정 예산과 잔액을 같은 금액만큼 감소시키되, 잔액이 음수가 되는 경우 추가 승인 대상으로 본다.
- 마감 예산 기간은 backend에서 예산 조정 또는 재오픈을 차단한다.

## 공급가액, 부가세, 총액

- 기본 통화는 KRW다.
- 공급가액은 부가세 제외 금액이다.
- 부가세는 과세 거래에 대해 공급가액의 10%로 계산한다.
- 총액은 `공급가액 + 부가세`다.
- 예산 차감 기준은 총액이다.
- 면세 또는 영세율 거래는 부가세를 0원으로 처리하되 증빙 유형에 면세/영세율 여부를 남긴다.
