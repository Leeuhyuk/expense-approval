# Feature Specification

작성일: 2026-07-05

## 화면별 기능

| 화면 | 핵심 기능 | 검증 기준 |
| --- | --- | --- |
| 대시보드 | KPI, 승인 추이, 부서별 지출, 긴급 결재, 최근 활동, 관련 화면 이동 | 주요 카드와 상세 보기 버튼이 업무 화면으로 이동한다. |
| 결제 요청 | 목록 검색/정렬/페이지, 상세 필터, 다운로드, 신규 요청, 임시 저장, 제출, 첨부 다운로드, 예산/결재선 변경 | 필수값과 예산 잔액을 검증하고 제출 완료 메시지를 표시한다. |
| 승인 관리 | 내 요청, 상세 필터, 다운로드, 일괄 승인, 승인/반려/보류, 첨부 다운로드, 결재선 현황 | 처리 권한과 사유 필수 입력을 검증하고 부분 실패/감사 로그 메시지를 표시한다. |
| 지급 관리 | 지급 목록, 상세 필터, 이체 파일 다운로드, 일괄 지급, 재처리, 계좌 재확인, 보류, 지급 일정 | 계좌 확인 완료 상태만 지급 실행하고 지급번호/감사 로그 메시지를 표시한다. |
| 예산 관리 | 기간/부서/상태/항목 필터, 다운로드, 예산 조정 입력, 카드/표 보기, 조정 이력, 연결 요청 | 사용률과 상태를 계산하고 승인 필요 여부를 표시한다. |
| 거래처 관리 | 검색, 상태/계좌/유형 필터, 다운로드, 추가, 수정, 비활성화, 계좌 재확인, 중복 검증, 증빙 파일 업로드, 지급 이력 탭 | 사업자번호와 계좌번호 중복을 차단하고 사업자등록증/통장사본/세금계산서 파일을 관리한다. |
| 보고서 | 유형 필터, 상세 필터, 검색/정렬/페이지, 보고서 생성, 상세 미리보기, 차트 연동, CSV/PDF 다운로드, 예약 발송 | 다운로드 파일이 생성되고 선택 보고서 미리보기가 표시된다. |
| 시스템 설정 | 탭별 섹션 전환, 승인 한도 직접 편집/삭제 영향 안내, 결재선 규칙, 권한 그룹 등록, 사용자 검색/권한, 부서 관리, 알림 테스트, 외부 연동 검증, 변경 이력 필터 | 변경 즉시 메시지와 변경 이력을 표시한다. |
| 즐겨찾기 | 자주 쓰는 메뉴, 저장 필터, 최근 사용, 입력 기반 추가, 순서 편집, 열기 라우트 이동, 삭제 undo, 사용자별 저장 | 중복 추가를 차단하고 연결 필터를 상세 패널에 표시한다. |

## 공통 기능

- API 계층은 `erpApi` 단일 진입점으로 mock/remote mode를 전환한다.
- 목록형 화면은 검색, 필터, 정렬, 페이지네이션, 로딩, 오류, 빈 상태를 가진다.
- 상태 변경은 업무 규칙과 권한 규칙을 먼저 검증한다.
- 화면 버튼은 상세 필터 패널, 다운로드, 패널 닫기, 결과 메시지, 감사 로그 기준을 우선 연결한다.
- 버튼 상태는 localStorage 기반 최근 조건 복원, 비활성 사유 tooltip, 삭제 확인/undo 경로를 제공한다.
- 파일, 보안, 예외 처리 기준은 각각 `docs/file-handling-rules.md`, `docs/security-exception-rules.md`를 따른다.
